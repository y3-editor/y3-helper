本目录包含 Y3-Helper VSCode 扩展的完整维护文档，供新加入的维护者学习使用。

## 文档目录

|------|----------|
| [01-项目架构总览.md](./01-项目架构总览.md) | 项目整体结构、内网版与公开版的关系、目录说明 |
| [02-编译与打包.md](./02-编译与打包.md) | 本地编译方式、Webpack 打包流程、打包为 .vsix 安装包 |

| [06-功能流程详解.md](./06-功能流程详解.md) | 启动游戏、调试附加、初始化项目、Excel导表、插件系统等关键功能的完整流程 |

## 快速开始

如果你是新加入的维护者，建议按以下顺序阅读：

1. **先看** [01-项目架构总览](./01-项目架构总览.md) — 了解项目全貌
2. **再看** [04-启动流程详解](./04-启动流程详解.md) — 理解代码从哪里开始执行
3. **然后看** [05-核心模块详解](./05-核心模块详解.md) 和 [06-功能流程详解](./06-功能流程详解.md) — 深入理解各功能
4. **最后看** [02-编译与打包](./02-编译与打包.md) 和 [03-发布流程](./03-发布流程（GitHub）.md) — 掌握如何发布版本
