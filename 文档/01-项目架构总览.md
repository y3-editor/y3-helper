# 01 - 项目架构总览

## 1. 项目简介

Y3-Helper 是一个 VSCode 扩展插件，用于辅助开发者使用 Y3 游戏编辑器。它提供了初始化项目、启动游戏、调试附加、物编数据管理、Excel 导表、插件系统、ECA 编译等功能。

---

## 2. 内网版与 GitHub 公开版的关系

本项目同时存在两个版本：

| 特性 | 内网版（公司仓库） | 公开版（GitHub） |
|------|-----------------|---------------|
| 仓库地址 | 公司内网 Git | https://github.com/y3-editor/y3-helper |
| 发布流程 | ❌ 不需要（手动打包） | ✅ 自动（GitHub Actions） |
| 代码同步 | 以内网版为主，定期推送到 GitHub | 从 GitHub 接受外部 PR |
| 功能范围 | 可能包含实验性/内部功能 | 与内网版保持一致 |

**同步方式：**

```bash
# 将内网改动推送到 GitHub
git remote add github https://github.com/y3-editor/y3-helper
git push github main
```

## 4. 技术栈

| 技术 | 说明 |
|------|------|
| TypeScript | 主要开发语言，编译目标为 ES2022 / Node16 模块 |
| Webpack 5 | 将所有 TS 模块打包为单一 `dist/extension.js` |
| VSCode Extension API | 提供树形视图、命令、语言服务等 UI 接口 |
| Node.js VM | 在安全沙箱中运行用户插件 JS 脚本 |
| ExcelJS | 读取用户的 Excel 导表文件 |
| @vscode/l10n | 国际化（中文/英文）支持 |
| GitHub Actions | CI/CD 自动构建和发布 |

## 5. 扩展的激活条件（`package.json`）

```json
"activationEvents": [
    "workspaceContains:y3/更新日志.md",
    "workspaceContains:y3/CHANGELOG.md",
    "onDebugResolve:y3lua"
]
```

即：当打开的工作目录包含 Y3 项目标志文件，或启动 `y3lua` 类型的调试时，扩展自动激活。
